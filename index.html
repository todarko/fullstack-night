PC Install

Download the installer for your system and install code editor


During Install

Select the following options during install to make it easier for opening projects directly in VSCode.


Install C# Extension

This will aid in writing code and syntax highlighting as well as debugging.


Open in VS Code

This will automatically install the extension and no reload is necessary.


Download and install git

Use default selections during install setup.

Run the following from a terminal to verify install. It will return your version number.

git --version

Download and install .net Core SDK

Select the correct installer for your system.

Run the following from a terminal to verify install. It will list sdk's installed.

dotnet --list-sdks

Download and Install PostgreSQL

We'll be connecting to the server and locally via this tool for Database actions.

Install with all defaults selected. This will include PostgreSQL, DB tools, pgAdmin
